# liberstar
A book review REST API to review books, kind of similar to goodreads.

## Details
  - Components User, Books, Reviews (YET)
  - CRUD operations
  - Input validation
  - SQL based
  - Authentication 
  - Role Based Authorization

## Get started

Run inside project directory
  ```console
  docker-compose up -d
  npm run migrate:latest
  npm run start
  ```
  
  Run tests
  ```console
  npm run test
  ```
